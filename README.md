# Spreadsheet Engine (React)

This repository contains a complete React project that implements a spreadsheet engine with
formula evaluation.

---

## Source Code
The full source code for the application is available in this repository.

Main files:
- `src/App.js` – Spreadsheet logic and formula evaluation
- `src/App.css` – Styling and responsive layout
- `src/index.js` – Application entry point

---

## Installation and Running the Project

### Prerequisites
-React.js
- Node.js
- npm

### Steps
npm install
npm start
